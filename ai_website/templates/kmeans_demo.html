What is Clustering?

K-Means is an unsupervised algorithm — it groups data points into k clusters without any labeled examples. The algorithm discovers structure in the data purely from the distances between points.

It's used in customer segmentation, image compression, anomaly detection, and document grouping.

No Labels Needed Distance-Based Centroid Method

The Algorithm

1 Initialize: Place k centroids using K-Means++ seeding (weighted random selection).
2 Assign: Each point is assigned to its nearest centroid by Euclidean distance.
3 Update: Each centroid moves to the mean position of all points assigned to it.
4 Repeat: Steps 2–3 repeat until no assignment changes (convergence).
centroid_j = mean of all x_i where assign(i) = j

Key Takeaways

K-Means++ seeding — Choosing the first centroid randomly then subsequent ones with probability proportional to distance² dramatically reduces bad initializations.
Always converges — K-Means is guaranteed to converge because each step (assign + update) can only decrease or maintain the total within-cluster variance, never increase it.
k is a hyperparameter — There's no single correct k. Use the elbow method (plot inertia vs k) or silhouette score to help choose a reasonable value.
Feature scaling matters — If features have very different scales (e.g., income vs age), standardize them first or the algorithm will be dominated by the larger-scale feature.
