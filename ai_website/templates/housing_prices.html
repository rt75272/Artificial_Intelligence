What is Regression?

Unlike classification (predicting a category), regression predicts a continuous number — like a price. Linear regression fits a weighted equation to historical data so that predictions minimize overall error.

price = w₁·sqft + w₂·beds + w₃·baths + w₄·year + b
Supervised Learning Continuous Output

How the Model Works

1 Features: Four input variables (sqft, bedrooms, bathrooms, year built) are collected.
2 Scaling: Features are standardized (mean=0, std=1) so no single variable dominates due to scale differences.
3 Weights: The model multiplies each scaled feature by a learned coefficient.
4 Sum: Weighted features are added with a bias term to produce the price estimate.

Key Takeaways

Feature scaling is essential — Without standardization, a large-valued feature like sqft would overshadow year_built in weight optimization.
Weights encode importance — The model learns which features matter most. Inspecting coefficients makes the model explainable.
Linear models are interpretable — Unlike neural networks, every prediction can be fully explained by the weighted sum formula.
More features ≠ better model — Adding irrelevant or correlated features can hurt accuracy through multicollinearity.
