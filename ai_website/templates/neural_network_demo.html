Layers of the Network

This model has four layers. Your drawing is transformed step-by-step as it passes through each one:

Input (Layer 0)784 pixels (28×28)
Hidden 1256 neurons + ReLU
Hidden 2256 neurons + ReLU
Output (Layer 3)10 neurons + Softmax

Each neuron computes a weighted sum of its inputs, then applies a nonlinear activation function.

Activation Functions

Without nonlinearity, stacking layers would be equivalent to a single linear transformation. Activation functions break this linearity and allow the network to learn complex patterns.

ReLU(x) = max(0, x)

ReLU simply zeroes out negative values, leaving positive values unchanged. It's fast to compute and avoids the vanishing gradient problem of older activations like sigmoid.

Softmax(xᵢ) = eˣⁱ / Σ eˣʲ

The output layer uses softmax, converting raw scores into probabilities that sum to 1.

Reading the Visualization

The neuron grids show activation values after each layer. A bright/white cell means a strongly activated neuron; a dark cell means near-zero or zero.

Key Takeaways

Depth enables abstraction — Early layers learn low-level features (edges, strokes), while deeper layers combine these into higher-level concepts (digit shapes). This hierarchical learning is why depth matters.
ReLU solves vanishing gradients — Older activations like sigmoid saturate (output near 0 or 1) and produce nearly-zero gradients, making deep networks hard to train. ReLU avoids this in its positive region.
Softmax gives probabilities — The final softmax layer converts raw scores into a proper probability distribution. The network is confident when one class has a value near 1 and the rest are near 0.
784 input features — Each 28×28 pixel image is flattened into a 784-dimensional vector. The model never "sees" the 2D structure — that's why convolutional networks (CNNs) perform better on images by explicitly modeling spatial locality.
